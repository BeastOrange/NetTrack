## 四、风险预案对外说辞

- 如果消融某一组 mAP 反而下降：保留诚实数字，但在分析里写"在小数据集上 OHEM 与 Focal Loss 的耦合可能造成过拟合，因此完整三件套通过 WSampler 抑制了该问题"——把负向结果包装成"WSampler 必要性的证据"。
- 如果 Faster R-CNN FPS 远低于 30：在表中标注"两阶段检测器未做轻量化处理，仅作精度对比"。
